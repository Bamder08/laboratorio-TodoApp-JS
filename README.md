# 🧪 laboratorio-TodoApp-JS

Aplicación de lista de tareas (TodoApp) desarrollada con JavaScript puro, enfocada en entender los fundamentos del manejo de estado, renderizado del DOM y persistencia de datos.

---

## 🚀 Deploy
https://laboratorio-todo-app.netlify.app

---

## 🧠 Objetivo del proyecto

Este proyecto fue creado como parte de mi proceso de aprendizaje para:

- Comprender cómo funciona el estado en aplicaciones frontend
- Manipular el DOM sin frameworks
- Implementar un store básico desde cero
- Persistir información usando localStorage

---

## ⚙️ Funcionalidades

- ✅ Crear nuevas tareas
- ✅ Eliminar tareas
- ✅ Marcar tareas como completadas
- ✅ Renderizado dinámico del DOM
- ✅ Persistencia de datos con localStorage

---

## 🧱 Tecnologías utilizadas

- JavaScript (Vanilla JS)
- Vite
- HTML
- CSS

---

## 🧩 Arquitectura del proyecto

El proyecto está estructurado siguiendo conceptos fundamentales del frontend:

- **Estado (state):** lista de tareas
- **Store:** objeto central que gestiona el estado y sus cambios
- **Render:** actualización del DOM en función del estado
- **Persistencia:** uso de localStorage para mantener la información

---

## 🔄 Flujo de la aplicación

1. El usuario interactúa con la UI (input, botones)
2. Se captura el evento
3. Se actualiza el estado
4. Se guarda en localStorage
5. Se renderizan los cambios en el DOM

---

## 📦 Instalación y uso

1. Clona el repositorio:
```bash
git clone https://github.com/Bamder08/laboratorio-TodoApp-JS.git
```

2. Instala las dependencias con el siguiente comando:
```bash
npm install
```

3. Ejecuta el siguiente comando para ver el proyecto en el navegador:
```bash
npm run dev
```
